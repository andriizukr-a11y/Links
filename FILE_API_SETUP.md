# Синхронізація через File API (локальний файл)

## Що це дає?

- ✅ Нотатки зберігаються в локальному файлі на вашому комп'ютері
- ✅ Автоматична синхронізація кожні 30 секунд
- ✅ Не потрібен GitHub або інтернет
- ✅ Повна контроль над місцем збереження файлу
- ✅ Простіше налаштування ніж GitHub Gist

## Крок 1: Вибір методу синхронізації

1. Відкрийте вкладку **Нотатки**
2. Натисніть кнопку **⚙** (налаштування) в правому верхньому куті
3. У модальному вікні виберіть **Локальний файл (File API)**

## Крок 2: Вибір файлу

1. Натисніть кнопку **"Вибрати файл"**
2. У діалозі вибору файлу:
   - Виберіть папку, де зберігатимуться нотатки
   - Введіть назву файлу (за замовчуванням: `tab-links-notes.json`)
   - Натисніть **"Зберегти"**
3. Браузер запросить дозвіл на доступ до файлу - дозвольте

## Крок 3: Налаштування синхронізації

1. Переконайтеся, що чекбокс **"Увімкнути синхронізацію з локальним файлом"** позначено
2. Натисніть **"Синхронізувати зараз"** щоб зберегти поточні нотатки
3. Натисніть **"Зберегти"**

## Крок 4: Завантаження нотаток з файлу

Якщо у вас вже є файл з нотатками:

1. Натисніть **"Вибрати файл"** і виберіть існуючий файл
2. Натисніть **"Завантажити з файлу"**
3. Підтвердіть, що хочете замінити поточні нотатки
4. Натисніть **"Зберегти"**

## Як це працює?

- **Автоматична синхронізація**: Кожні 30 секунд, якщо є зміни
- **Індикатор статусу**: Кольорова точка біля кнопки налаштувань:
  - 🔵 Синій - синхронізація в процесі
  - 🟢 Зелений - успішно синхронізовано
  - 🔴 Червоний - помилка синхронізації
  - ⚪ Сірий - очікування змін

## Переваги File API

- **Приватність**: Дані зберігаються тільки на вашому комп'ютері
- **Швидкість**: Немає затримок на мережу
- **Простота**: Не потрібні токени або облікові записи
- **Надійність**: Файл завжди під вашим контролем

## Недоліки File API

- Синхронізація працює тільки на цьому комп'ютері
- Потрібен дозвіл браузера на доступ до файлу
- Не можна отримати доступ з іншого браузера або пристрою

## Переключення між методами синхронізації

Ви можете змінити метод синхронізації в будь-який час:

1. Відкрийте налаштування (⚙)
2. Виберіть інший метод синхронізації
3. Налаштуйте параметри
4. Натисніть **"Зберегти"**

## Усунення проблем

### Помилка "Доступ до файлу не надано"
- Переконайтеся, що ви вибрали файл перед синхронізацією
- Спробуйте вибрати файл ще раз

### Помилка при збереженні файлу
- Перевірте, що у вас є права на запис в папку
- Переконайтеся, що файл не відкритий в іншій програмі
- Спробуйте вибрати іншу папку

### Нотатки не синхронізуються
- Перевірте, що індикатор показує активність
- Відкрийте консоль браузера (F12) і перевірте наявність помилок
- Спробуйте натиснути **"Синхронізувати зараз"** вручну
